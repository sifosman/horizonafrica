Server Management Commands:
- Restart n8n: cd /opt/n8n && docker compose restart
- Update n8n: cd /opt/n8n && docker compose pull && docker compose up -d
- View logs: cd /opt/n8n && docker compose logs -f
- View n8n logs only: cd /opt/n8n && docker compose logs -f n8n
- View Caddy logs: cd /opt/n8n && docker compose logs -f caddy
- Stop all services: cd /opt/n8n && docker compose down
- Start all services: cd /opt/n8n && docker compose up -d
