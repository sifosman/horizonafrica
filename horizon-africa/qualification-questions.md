# Horizon Africa — Lead Qualification Questions & Product Recommendation Logic

## Qualification Questions

### Q1. How many people will be using the internet?

Possible answers:
- 1 Person
- 2-4 People
- 5+ People

### Q2. What do you mainly use the internet for?

Possible selections (can be multiple):
- Email
- Browsing
- Netflix
- YouTube
- Gaming
- Working from Home
- CCTV
- Smart Home Devices
- Business
- Online School
- Video Calls

## Lead Scoring Logic

### HOT Lead
- User provides full name + contact number + address
- User confirms product interest and asks about installation/pricing
- User asks "how do I sign up" or "can you send someone"
- User responds to qualification questions with clear intent

### WARM Lead
- User engages with qualification questions but hasn't provided all contact details
- User asks about pricing but is comparing options
- User provides partial information (name but no address, etc.)

### COLD Lead
- User asks a general question and doesn't engage further
- User asks about pricing and says it's too expensive
- User doesn't respond to follow-up questions
- User is just browsing / gathering information
